## NofalSeo Task

### How To Run

- Download this code


- Install composer

```bash 
composer install
 ```

- Install npm

```bash 
npn install & npm run dev
 ```

- Edit .env file


- Run migration

```bash 
php artisan migrate
 ```

- Run The App

```bash 
php artisan serve
 ```
